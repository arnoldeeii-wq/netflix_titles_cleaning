# Netflix Titles Cleaning Project

# Overview
Cleaned and standardized Netflix titles dataset using SQL:
- Added row numbers
- Standardized missing values
- Prepared dataset for Excel pivot analysis

# Tools
- MySQL
-  Google Sheets

# Insights
- Titles by country
- Genre distribution by rating
- Titles added over time
